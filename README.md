# Wikipedia Web Scraping with Python

A simple Python web scraping project that extracts data from Wikipedia tables using `BeautifulSoup` and `requests`.

## Features

- Fetches webpage content using Requests
- Parses HTML with BeautifulSoup
- Extracts table headers and rows
- Works with Wikipedia sortable tables
- Beginner-friendly web scraping example

## Technologies Used

- Python
- BeautifulSoup4
- Requests
- Jupyter Notebook
